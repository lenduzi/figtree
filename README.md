# Figtree CRM

A lightweight CRM built for solo founders and small startups.

Live product: [figtreecrm.com](https://figtreecrm.com)

## What it is

Figtree is a simple CRM focused on the essentials:
- contacts
- follow-ups
- pipeline management
- lightweight outreach ops
- a built-in productivity layer

The goal was to build a CRM that feels clean, fast, and actually usable for a solo operator.

## Why I built it

Most CRMs feel bloated, expensive, or overbuilt for a one-person business.
I wanted something simpler, so I built Figtree.

## Build process

Figtree started with an initial Lovable scaffold/prototype and was then extended, refined, and developed further through custom repo work using Codex and manual iteration.

So while the project was not built 100% from scratch line-by-line on day one, the product direction, feature set, structure, and ongoing implementation were actively developed by a user like you.

## Features

- Follow-up dashboard
- Contact management
- Pipeline view
- Outreach operations workspace
- Project/resource organization
- Lightweight productivity layer

## Tech stack

- React / TypeScript
- Vite
- Tailwind CSS
- Supabase (optional auth + sync)
- Lovable for early scaffolding
- Codex for repo-level development support
